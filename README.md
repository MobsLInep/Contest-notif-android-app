# Bolt Expo Nativewind (Flutter Replica)

This is a 100% replica of the original Expo/React Native app, rebuilt in Flutter.

## Getting Started

1. Install Flutter: https://docs.flutter.dev/get-started/install
2. Get dependencies:
   ```
   flutter pub get
   ```
3. Run the app:
   ```
   flutter run
   ```

## Features
- Theming (light/dark)
- Codeforces contest list and countdown
- User profile and stats
- Settings and notifications (UI only)
- Error and loading states

All UI, logic, and assets are replicated from the original Expo app.
